MAPACHE

Inicio

Mapache es un servidor web que permite compartir páginas web a través de Internet.

Ventajas:

* Es de fácil configuración gracias a la interfaz gráfica creada para ese propósito.

* Permite hostear multiples sitios simultaneamente utilizando distintos puertos.

* Es de fácil instalación

En esta página usted podrá encontrar información necesaria para instalar, configurar y utilizar este software. Para ello visite la sección Ayuda de este sitio.
