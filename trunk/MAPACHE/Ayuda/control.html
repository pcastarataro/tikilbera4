Ejecutando el servidor

Tenemos tres operaciones posibles sobre nuestro servidor.

A continuación detallamos cada una de ellas.

3.1. Ejecutando el servidor

- Iniciar Servidor: esta operación toma la configuración actual de la aplicación y con ella inicia un servidor que respete esos valores de configuración.

Es importante no olvidar que si se cierra la aplicación sin detener el servidor, este seguira ejecutandose. Solo se detiene si se presiona en el botón detener.

3.2. Reiniciando el servidor

- Reiniciar servidor: esta operación renicia el servidor que estaba ejecutandose con el puerto de control indicado por la configuración actual. Puede ser que tarde algunos segundos en reiniciar.

3.3. Deteniendo el servidor

- Detener servidor: esta operación detiene el servidor. Si había clientes conectados, los desconecta y luego se apaga. Puede ser que tarde algunos segundos en apagar el mismo.
